# Daily Learning Streak Tracker

A full stack web application that helps students maintain
daily learning consistency by tracking study streaks.

## Features

- Track daily study streak
- Prevent duplicate study entries
- Visual streak calendar
- Study analytics chart
- Dark mode support
- Responsive modern UI

## Tech Stack

Next.js  
TypeScript  
Tailwind CSS  
API Routes  
Local JSON storage

## Installation

npm install  
npm run dev

## Deployment

Deployed using Vercel.
